
# Weather station

Weather station is the server that detects the weather conditions and send it to it's app Weatherapp




## Tech Stack

**Client:** c++, Arduino

**Server:** Firebase


## Prerequisites
- Esp32 board 
- Arduino IDE
- DHT11 sesnsor 
- BMP085 sensor
